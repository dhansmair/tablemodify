Introduction

tablemodify (tm) is a thin javascript plugin, easy to use and offers a lot of useful functionalities to manipulate HTML-tables. tm is based on a core object which contains basic modules. Very soon, it can also be extended by your own modules! You can choose which modules you want to activate. version Please note that the current verion (v0.5) is not a stable version yet. There will be many changes and extensions in the next time. Furthermore, the collection of basic modules will grow. Browser support supported by all browsers which support css flexboxes. It was successfully tested in Chrome, Firefox and IE11. see http://caniuse.com/#feat=flexbox Tablemodify is not optimized for mobile browsers yet.

Basic setup

download the files tablemodify.js and tablemodify.css from gitHub. Include them into your html file: 

     <script src='path/to/tablemodify.js'></script>
     <link href='path/to/tablemodify.css' rel='stylesheet' />

Initialize the Tablemodify-plugin like this (after DOM is loaded!): 

    <script>
        function domReady() {
            var tm = new Tablemodify('#tm-demo', settings);
        }

        window.addEventListener('DOMContentLoaded', domReady);
    </script>

the passed js-object settings may have the following properties: 


    var settings = {
        /*
            your table will be wrapped in a div. You can add a custom id to it.
            (default: "tm-container")
        */
        containerId: 'myCustomContainer',
        /*
            most important part of the settings.
            All modules are optional.
            To run a module, add it's name to the following object. the values are js-objects again.
            To run a module with default values, you can just pass an empty object, for example: "zebra: {}",
            The specific module settings will be explained in the accompaniyng chapter.
        */
        modules: {
            zebra: zebraSettings,
            fixed: fixedSettings,
            sorter: sorterSettings,
            columnStyles: columnStylesSettings
        }
    };

module columnStyles

Use columnStyles to set the CSS for cells in each column. 


    var columnStylesSettings = {
        /*
            this styles all cells.
            you can add any CSS property that work for html td and th elements.
            Notice that the values will be overwritten by specific styling of single columns!
        */
        all: {
            'text-align': 'left',
            'padding': '5px',
            'min-width': '150px',
            'max-width': '300px'
        },
        /*
            attach styles to cells with these indices.
            First index is 0.
        */
        0: {
            'text-align': 'right',
            'font-weight': 'bold'
            // ...
        },
        1: {
            'background-color':'rgba(0,0,0,0.5)'
            // ...
        }
        // ...
    };

module fixed

a super useful module and basically the reason why i built Tablemodify. In my opinion there is no good solution for fixed table headers on the internet, most of them have problems this module can deal with:

  • window resizing
  • horizontal scrolling
  • cross-browser functionality
  • compatibility with a table-sorter

it is possible to fix both thead and tfoot elements. The settings are simple: 


    var columnStylesSettings = {
        /*
            these are the defaults.
            To fix thead or tfoot, set it to true
        */
        fixHeader: false,
        fixFooter: false
    };

module sorter

This module sorts the rows of your tbody, ascending or descending, sorted by a single column or even by multiple columns. For multisort, press shift and click another head cell. Still press shift and click a selected cell again to invert the order. Please keep in mind that js is generally rather slow and not made for handling big data amounts. Handling about 3000 rows works fine. 5000 still work, but it will take some time to sort and re-render, so bigger tables should be sorted on the backend, for example via php. 


    var sorterSettings = {
        /*
            these are the defaults:
        */
        initial:[0, 'asc'],         // initial sorting. Array with this structure: [index, 'asc' || 'desc']
        enableMultisort: true,      // enable/disable multisort functionality
        headers: {
            all: {
                enable: true,
                parser: 'intelligent'
            },

            /*
                custom settings for single columns:
            */
            0: {
                enable: false           // disable sorting
            },
            1: {
                parser: 'germanDate'    // use another parser for this column
            }
        },
        /*
            adding own parsers. Read below how it works.
        */
        customParsers: customParsers
    };

so, what about the parsers? There are some available by default:

parsername notice description
"intelligent" default differentiates automatically between strings and numbers. recommended for mixed types.
"string" sorts in alphabetic order. efficient, recommended for pure strings
"numeric" sorts numbers (also floats). efficient, recommended for numbers
"germanDate" dd.mm.yyyy
d.mm.yyyy
d.m.yyyy 		sorts by german date format. days between 1 and 9 can be written as "1" od "01",
so do months.

A parser is simply a compare function which is passed to the Array.sort() method. here you can read more about compare functions If you want to define your own parsers, add them like this: 


    var customParsers = {
        myParser: function(a, b) {
            /*
                parameters a, b: strings of two cell-contents to compare.

                if a > b, return a positive number,
                if a < b, return a negative number,
                if a = b, return 0. (<- this is important for multisort!)
            */
            // for example, this is the implementation of parser "string"
            if (a > b) return 1;
            if (a < b) return -1;
            return 0;
        }
    };

now we have defined our own parser and we can use it by the string "myParser".

module zebra

A small module to add stripes to your table. 


    var zebraSettings = {
        // these are the default settings:
        even: '#dbe7f0',
        odd: '#ffffff'
    };

yeah, that's it.
